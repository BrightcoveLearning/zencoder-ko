의미

이 오류는 Access Denied 또는 File Not Found 이외의 이유로 파일을 다운로드 할 수 없을 때 발생합니다. 서버가 응답 코드를 반환하면 오류 메시지에이를 표시하여 발생한 상황을 확인하고 파일에 필요한 업데이트를 수행 할 수 있습니다.

이 작업을 다시 시도 하시겠습니까?

이 오류가있는 작업은 다시시도할 수 없습니다. 작업 재 시도에 대해 자세히 알아보세요.
