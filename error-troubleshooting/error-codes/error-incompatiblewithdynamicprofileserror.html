    동적 프로필과 호환되지 않습니다오류

    이 항목에서는 Zencoder API 오류에 대해 자세히 설명합니다IncompatibleWithDynamicProfilesError .

    의미

    입력 미디어가 CAE 동적 프로파일 생성과 호환되지 않을 때 이 오류가 발생합니다. 입력 미디어 파일에는 오디오 및/또는 이미지 데이터만 있고 비디오 트랙이 포함되어 있지 않음을 의미할 수 있습니다. 자세한 정보를원하시면 Zencoder 지원팀에 문의하세요 .

    이 작업을 다시 시도하시겠습니까?

    이 오류가 발생한 작업은 다시시도해서는 안됩니다. 작업 재시도에 대해 자세히 알아보십시오.
