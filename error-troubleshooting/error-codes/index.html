오류 코드

이 주제에서는 Zencoder API 요청에 대한 HTTP 상태 코드 요약 및오류 코드 색인 Zencoder API에서 반환합니다. 다음 중 하나를 클릭하십시오. 오류 코드자세한 내용을 보려면.

HTTP 상태 코드

다음 표에는 오류 발생시 Zencoder API 요청에 대한 HTTP 응답 코드가 나열되어 있으며 가능한 경우 가장 가능성이 높은 원인을 나타냅니다.

HTTP 응답 코드
상태 코드 가능한 설명
200 OK 새 자산을 생성하지 않는 작업에 대한 표준 성공 대응
201 CREATED 새 자산을 생성하는 요청에 대한 표준 응답
204 NO CONTENT 응답이없는 요청에 대한 표준 응답
400 BAD REQUEST 요청 본문의 잘못된 JSON
401 UNAUTHORIZED 누락되거나 잘못된 API 키
402 PAYMENT REQUIRED API 요청이 결제가 설정되지 않은 계정에서 Zencoder 통합 모드 https://app.zencoder.com/docs/api/accounts/integration을 끄려고 할 때 반환됩니다. www.zencoder.com으로 가서 지불 계획을 선택해야 합니다.
403 FORBIDDEN 초과된 요청에대한 속도 제한 또는 너무 여러 번 재시도된 리소스에 대한 재시도/재제출 API 요청에 대한 응답으로
404 NOT FOUND 잘못된 요청 URL 또는 잘못된 / 이전 작업 ID
406 NOT ACCEPTIBLE API를 통해 비 AWS 자격 증명 생성을 시도한 경우 반환 됨
409 CONFLICT 일반적으로 너무 오래된 작업을 다시 시도 / 다시 제출하려고 할 때만
422 UNPROCESSIBLE ENTITY API 요청을 확인할 수 없을 때 반환됩니다.
500 INTERNAL SERVER ERROR Zencoder의 일시적인 문제 또는 요청에 문제가 있습니다.
503 SERVICE UNAVAILABLE Zencoder를 일시적으로 사용할 수 없습니다. 예정된 유지 관리 중일 수 있습니다.
504 GATEWAY TIMEOUT Zencoder 서비스가 다운 된 경우 반환됩니다.
