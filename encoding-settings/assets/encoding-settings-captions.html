개요

Zencoder는 지원되는 형식에 대해 입력에서 출력으로 캡션을 자동으로 전달합니다. 트랜스 코딩 중에 출력에 추가 할 캡션 파일 (SCC 형식)을 지정할 수도 있습니다.

비디오 트랙에 다중화된 캡션은 ~ 아니다지원.

캡션에 지원되는 입력 형식:

  • CEA-608 형식의 캡션 트랙이있는 MP4 / M4V 비디오 파일
  • 캡션_URL 옵션으로 지정된 VTT, SCC, DFXP 또는 SAMI 파일

캡션에 지원되는 출력 형식:

  • MP4 / M4V 파일 (CEA-608 형식의 캡션 트랙 포함)
  • HLS 출력 (TS 형식의 H.264 비디오가있는 세그먼트 출력)
  • SCC 및 DFXP 형식은캡션유형출력에서도 지원됩니다.

HLS의 경우 적절한 매니페스트 캡션을 얻으려면 캡션을 출력에 매핑해야합니다. 예를 들면 다음과 같습니다.

{ "label": "Captions", "type": "captions", "segmented": true, "public": true, "filename": "captions.m3u8", "caption_url": "PATH_TO_CAPTION_FILE_HERE", "streaming_delivery_format": "hls" } , { "filename": "master_playlist.m3u8", "type": "playlist", "captions": [{ "path": "Captions/captions.m3u8", "language": "en", "name": "english" } ], “스트림”: [ { "path": "video/video-1800.m3u8", "source": "video-res1", "audio": "64k_audio" } ]}

참고: 현재 캡션 변환에는 다음 제한 사항이 적용됩니다.

DFXP 입력의 경우:

  • 모든 특수 문자에는 UTF8 인코딩이 필요합니다.
  • 첫 번째 언어 만 변환됩니다 (첫 번째 DIV 태그).
  • 애니메이션, 레이아웃 및 지역 기능은 지원되지 않습니다.
  • 스타일은 인라인이거나 ID에 의해 참조될 수 있지만 tts:textAlign 속성만 적용됩니다.
  • 중첩 된 DIV 및 P 태그는 무시됩니다.
  • SPAN 태그는 무시됩니다.
  • BR 태그와 함께 명시 적 구분을 삽입 할 수 있습니다.
  • 언제든지 하나의 캡션 만 화면에 표시 될 수 있습니다 (이후 캡션은 이전 캡션이 종료 됨).

SAMI 입력의 경우:

  • 나열된 첫 번째 언어 만 변환됩니다 (첫 번째 클래스 선택기 스타일).
  • 인라인 스타일은 지원되지 않습니다.
  • 레이아웃 관련 스타일 지정 태그 (테이블, div 등)는 지원되지 않습니다.
  • 별도의 P 태그를 사용하여 명시 적 구분을 삽입 할 수 있습니다.

caption_url

캡션_URL:문자열

API 버전: V1, V2

상위: 출력

유효한 값: 출력에 포함 할 SCC, DFXP 또는 SAMI 캡션 파일에 대한 유효한 URL

호환 가능한 작업 유형: 모두

예: http://example.com/captions.scc

기술:

출력에 포함 할 SCC, DFXP 또는 SAMI 캡션 파일의 URL입니다.

Zencoder는 여러 출력 비디오 형식에 대한 캡션을 지원합니다.

  • MP4 / M4V 파일 (CEA-608 형식의 캡션 트랙 포함)
  • HLS 출력 (TS 형식의 H.264 비디오가있는 세그먼트 출력)

 

참고: 현재 트랜스 먹싱 된 출력에서는 캡션이 지원되지 않습니다.

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "caption_url": "http://example.com/captions.scc" } ]}

참조: 건너뛰기및준비_포_세그먼트 지정

skip_captions

건너뛰기_캡션:부울

API 버전: V1, V2

상위: 출력

기본값:거짓

유효한 값:참 또는 거짓

호환 가능한 작업 유형: 모두

예: true

기술:

출력 파일에 캡션을 추가하거나 전달하지 마십시오.

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "skip_captions": "true" } ]}

참조: 캡션_URL및준비_포_세그먼트 지정
