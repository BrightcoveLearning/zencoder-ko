start_clip

시작_클립:문자열

API 버전: V1, V2

상위: 출력

유효한 값: 시간 코드 ( HH:MM:SS.S ) 또는 초 수를 나타내는 양수

호환되는 작업 유형: vod

예:

  • 00:00:10 .0
  • 01:31:08 .3
  • 60
  • 12.5

기술:

특정 시간에 시작하여 원본 비디오의 하위 클립을 만듭니다. HH:MM:SS.S형식의 시간 코드이거나 시작할 초 수를 나타내는 십진수 값일 수 있습니다. 실제 시작 시간은 비디오 내의 키 프레임 배열에 따라 완벽하게 정확하지 않을 수 있습니다.

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "start_clip": "00:00:10.0" } ]}

참조: 클립_길이

clip_length

클립_길이:문자열

API 버전: V1, V2

상위: 출력

유효한 값: HH:MM:SS.S 형식의 지속 시간또는 초 수를 나타내는 양수

호환 가능한 작업 유형: 모두

예:

  • 00:00:10 .0
  • 01:31:08 .3
  • 60
  • 12.5

기술:

특정 길이의 하위 클립을 만듭니다. 이 값은 HH:MM:SS.S형식의 지속 시간이거나 클립의 길이를 초 단위로 나타내는 십진수일 수 있습니다. start_clip과 결합하여 비디오 중간에 하위 클립을 만들 수 있습니다. start_clip이 제공되지 않으면 비디오의 시작 부분에서 클립이 시작됩니다. 실제 클립 길이는 비디오 내의 키 프레임 배열에 따라 완벽하게 정확하지 않을 수 있습니다. 또한 clip_length는 기간이며 하위 클립이 끝나야하는 미디어 파일의 시간 코드가 아닙니다.

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "clip_length": "00:00:10.0" } ]}

참조: 스타트_클립
