최소 크기

min_size:String

API 버전: V2

상위: 조건문

호환 가능한 작업 유형: 모두

예:

  • 640x480

기술:

소스의 크기가 요청 된 크기보다 작 으면 출력을 건너 뜁니다.

치수는 소스의 높이와 너비를 곱하고 제공된 치수의 곱과 비교하여 계산됩니다.

참고: 비디오 메타 데이터의 특성과 비디오 유형 간의 불일치로 인해 일부 소스 파일의 크기가 검사 중에 잘못 읽혀져 조건이 잘못 적용되거나 무시 될 수 있습니다.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "skip": {
        "min_size": "640x480"
      }
    }
  ]
}

참고: 오디오 메타 데이터의 특성과 오디오 유형 간의 불일치로 인해 일부 소스 파일의 비트 전송률이 검사 중에 잘못 읽혀져 조건이 잘못 적용되거나 무시 될 수 있습니다.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "skip": {
        "min_audio_bitrate": 128
      }
    }
  ]
}

require_audio

require_audio:Boolean

API 버전: V2

상위: 조건문

기본값:거짓

유효한 값:참 또는 거짓

호환 가능한 작업 유형: 모두

예: true

기술:

소스 파일에 오디오 트랙이없는 경우 출력을 건너 뜁니다.

소스에 오디오 트랙이없는 경우 출력을 건너 뛰려면이 옵션을 true로 설정하십시오.

참고: 미디어 메타 데이터의 특성과 미디어 유형 간의 불일치로 인해 일부 소스 파일은 검사 중에 메타 데이터가 잘못 읽혀져 조건이 잘못 적용되거나 무시 될 수 있습니다.


  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "skip": {
        "require_audio": true,
        "min_audio_bitrate": 64
      }
    }
  ]
}
