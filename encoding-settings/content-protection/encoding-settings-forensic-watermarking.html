소개

포렌식 워터마킹은 불법 복제 및 무단 콘텐츠 공유로부터 프리미엄 콘텐츠를 보호하는 데 도움이 됩니다. 이 기능은 또한 콘텐츠 유출의 원인을 신속하게 식별하여 조치를 취할 수 있도록 도와줍니다. 포렌식 워터마킹 사용에대한 요구 사항에 대한 자세한 내용은 포렌식 워터마킹을 참조하십시오.

포렌식 워터마크

forensic_watermark:String

API 버전: V2

상위: 출력

유효한 값: "아", "비"

하나의 출력과 두 번째 출력에AB대해 로 설정해야 합니다.

호환되는 작업 유형: vod

예: ㅏ

기술:

Zencoder에서 포렌식 워터마킹으로 출력을 생성하려면 비디오당 2개의 출력을 지정해야 합니다. NS그리고 NS각 출력의 세그먼트를 서로 다른 시퀀스로 혼합하여 재생을 위해 다양한 워터마크가 표시된 "비디오"를 생성할 수 있습니다(예: 아바바바, 아바바바바, 바바바바 등).

참고 사항:
  • 5분보다 짧은 동영상에 포렌식 워터마킹을 추가할수 없습니다.
  • segment_seconds로설정해야 합니다. 2그렇지 않으면 포렌식 워터마킹이 추가되지 않습니다.

forensic_watermark_stub_mode

forensic_watermark_stub_mode:Boolean

API 버전: V2

상위: 출력

유효한 값:참, 거짓

기본값: false

호환되는 작업 유형: vod

예: true

기술:

NS 통합 테스트 목적으로 사용 Zencoder에서 포렌식 워터마킹으로 출력을 생성합니다. 와trueforensic_watermark함께 로 설정하면 이 플래그로 인해 표시되는 워터마크가 생성됩니다 (포렌식 워터마킹이 성공적으로 추가되고 있는지 확인합니다.)

테스트가 완료되면forensic_watermark_stub_mode플래그를 생략하거나 로 설정하여 작업을 다시 실행하여 보이지 않는 출력을 가져와야 합니다. false워터마크.
