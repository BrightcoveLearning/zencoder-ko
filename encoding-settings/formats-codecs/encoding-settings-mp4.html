힌트

hint:Boolean

API 버전: V1, V2

상위:출력

기본값:거짓

유효한 값:참 또는 거짓

호환 가능한 작업 유형: 모두

예: true

기술:

RTP / RTSP 용 MP4 파일 힌트를 켭니다. 기본 mtu_size는 1450바이트입니다. mtu_size옵션을 사용하여 변경할 수 있습니다.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "hint": true
    }
  ]
}

mtu_size

mtu_size:Integer

API 버전: V1, V2

상위:출력

기본: 1450

유효한 값: 100에서 5000 사이의 정수

호환 가능한 작업 유형: 모두

예: 2000

기술:

MP4 힌팅을위한 MTU 크기 (바이트)입니다.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "mtu_size": 2000
    }
  ]
}
