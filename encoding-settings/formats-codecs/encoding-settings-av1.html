인코딩 권장 사항

AV1은 Alliance of Open Media Video의 개방형, 로열티가없는 차세대 비디오 인코딩 형식입니다. HEVC, VP9 및 H.264 코덱에 비해 훨씬 낮은 비트 전송률로 우수한 품질을 제공 할 수 있습니다.

자세한 내용은 AV1 가이드를참조하고 현재 지원의제한 사항에특히 주의하십시오.

이 페이지에 있는 설정 외에도 다른 일반 비디오 설정을 포함할 수 있지만 다음 설정은 ~ 아니다현재 지원:

  • video_reference_frames
  • video_bframes
  • max_video_bitrate
  • quality
  • crf
  • decoder_bitrate_cap

또한해상도 설정의 경우 ( size또는width및 사용 여부에 관계 없음) height ) 너비와 높이는 모두 8의배수여야 합니다.

video_codec_level

video_codec_level:String

API 버전: V2

상위: 출력

유효한 값:

  • 2.0
  • 2.1
  • 3.0
  • 3.1
  • 4.0
  • 4.1
  • 5.0
  • 5.1
  • 5.2
  • 5.3
  • 6.0
  • 6.1
  • 6.2
  • 6.3

호환 가능한 작업 유형: 모두

예: 4.1

기술:

비트 전송률 및 코딩 트리 단위 (CTU) (AV1)를 제한합니다.

이 설정은 허용되는 값이 다른 다른 코덱에 사용할 수 있습니다. 비디오 인코딩 설정을 참조하십시오 .

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_level": "5.1"
    }
  ]
}

video_codec_profile

video_codec_profile:String

API 버전: V2

상위: 출력

유효한 값: main , high , professional

기본값: main

호환 가능한 작업 유형: 모두

예: 4.1

기술:

인코딩 프로필을 설정합니다.

이 설정은 허용되는 값이 다른 다른 코덱에 사용할 수 있습니다. 비디오 인코딩 설정을 참조하십시오 .

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_profile": "high"
    }
  ]
}
