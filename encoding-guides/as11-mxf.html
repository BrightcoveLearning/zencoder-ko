개요

AS-11 출력을 만들려면format_preset매개 변수를 “as11_hd”로 설정하십시오. 현재 오디오는 2채널 PCM 오디오로 제한되며 최대 샘플 속도는 48Khz입니다. AVC-intra 출력에 대한 자세한 내용은여기 문서를 참조하십시오 .

  • 포맷_프리셋 : AE AVCI 100 MXF 출력의 경우 이 값을 as11_hd로 설정합니다.
  • 오디오_샘플_속도 : 기본값은 설정되어 있지 않습니다. 48000으로 제한됩니다.

예제 설정

AS-11 요청 예

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "format_preset": "as11_hd",
        "audio_sample_rate": 48000
    }
}
