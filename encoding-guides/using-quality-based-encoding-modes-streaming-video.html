소개

스트리밍을 위해 비디오 출력을 인코딩할 때는품질설정만 사용하지 않는 것이 좋습니다 (CRF 인코딩 모드 사용). 품질 타겟팅 모드를 선호하는 경우 VBV 설정을 사용하여 스트리밍 형식 사양을 준수하도록 비트 전송률의 변형을 제한하는 것이 좋습니다.

VBV 설정

  • 디코더 _ 비트 레이트 _ 캡
  • 디코더 _ 버퍼 _ 크기

고려해야 할 기타 권장 설정

  • video_reference_frames
  • video_bframes

예

다음은 VBV 제한이있는 품질 대상 인코딩의 예입니다.

{
    "type":"segmented",
    "width":640,
    "height":360,
    "quality":3,
    "video_codec_profile":"main",
    "video_reference_frames": 4,
    "video_bframes": 3,
    "decoder_bitrate_cap": 1440,
    "decoder_buffer_size": 1920,
    "keyframe_rate": 0.5,
    "max_frame_rate": 30,
    "audio_quality":5,
    "segment_seconds":10
}

메모

  • 참조하십시오 HTTP 라이브(분할) 스트리밍 및 재생 목록그리고단편화된 MP4를 사용한 HTTP 라이브 스트리밍 HLS 출력 및 재생 목록에 대한 자세한 내용은
  • DASH 플레이리스트에대한 자세한 내용은 DASH 스트리밍 및 재생 목록을 참조하십시오.
