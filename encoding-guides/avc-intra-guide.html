AVC-Intra 개요

AVC-Intra는 AVC-Intra 50 및 AVC-Intra 100의 두 가지 사전 설정을 정의합니다. AVC-인트라 50은 50mbit/s의 비트 전송률로 인코딩되고 AVC-인트라 100은 100mbit/s로 인코딩됩니다. video_codec_preset매개 변수를 사용하여 AVCI 출력을 정의합니다. AS-11 출력 생성에 대한 자세한 내용은여기 에서 설명서를 참조하십시오 .

  • 비디오_코덱_프리셋 : 출력에 대한 AVC-Intra 사전 설정을 설정합니다. 값: avci_50, avci_100.

예제 설정

AVC-Intra 50 요청 예

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_50"
    }
}

AVC-Intra 100 요청 예

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_100"
    }
}
