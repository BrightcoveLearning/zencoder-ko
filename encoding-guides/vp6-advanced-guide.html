VP6 개요

VP6는 Flash Video에 널리 사용되는 비디오 코덱입니다. 품질은 일반적으로 H.264 또는 VP8보다 나쁘지만 여전히 존경 할 만하 며 Flash Player의 확산으로 호환성이 뛰어납니다. 따라서 H.264 또는 VP8이 많은 사람들에게 더 나은 옵션 일 수 있지만 VP6에 대한 합법적 인 사용 사례는 여전히 있습니다.

VP6은 FLV 파일 컨테이너에 배포되므로 출력 파일 이름을 지정하는 경우.flv 확장자를 사용하십시오.

AAC 및 MP3 오디오 코덱은 FLV 컨테이너에서 VP6과 함께 유효합니다.

VP6을 사용하려면 video_codec API 옵션을 “vp6"로 설정하십시오.

VP6 전용 옵션

vp6_템포럴_다운_워터마크 : 임시 다운 워터 마크 백분율에 대한 코덱 매개 변수입니다. 인코더가 프레임 삭제를 시작할 수있는 나머지 데이터 속도 버퍼의 백분율을 지정합니다. vp6_temporal_리샘플링이 활성화된 경우에만 사용됩니다. 유효한 값은 0-100입니다. 기본: 20 비율이 클수록 요청 된 비디오 비트 전송률을 달성하기 위해 프레임이 삭제 될 가능성이 높아집니다. 그 반대도 사실입니다.

vp6_임시 리샘플링 : 시간적 리샘플링을위한 코덱 매개 변수입니다. 특히 어려운 지역에서 활성화되면 인코더는 목표 데이터 속도를 달성하기 위해 프레임을 삭제합니다. vp6_temporal_down_watermark의 값에 따라 인코더가 프레임 드롭을 시작할 수 있는 데이터 레이트 버퍼의 백분율이 결정됩니다. 부울을 허용합니다. 낮은 픽셀 당 비트 수를 사용하는 파일에서는 기본값이 "true"이고 픽셀 당 비트 수가 많은 파일에서는 "false"입니다.

참고: constant_bitrate 모드를 사용하는 경우 이 옵션은 강제로 “true”로 설정됩니다. 결과를 향상 시키려면 vp6_temporal_down_watermark를 조정하십시오.

vp6_언더슈트_pct : 약간 더 낮은 데이터 속도를 목표로하는 출력을 생성하여 어려운 섹션을 개선하기 위해 버퍼에 사용할 수있는 비트가 있습니다. 이것은 백분율이므로 유효한 값은 0-100입니다. 기본값은 90입니다.

vp6_profile : “vp6s” 또는 “vp6e” 중 하나입니다. vp6e는 일반적인 용도로 설계되었습니다. vp6s는 고해상도/데이터 레이트 비디오에 최적화되어 있습니다. 디코딩 복잡성을 줄여 리소스가 제한된 컴퓨터에서 재생할 수 있습니다. 기본값은 vp6e입니다.

vp6_압축 모드 : 출력 품질과 비교하여 인코딩 속도에 영향을줍니다. "좋은"은 속도와 품질의 균형을 유지합니다. "최상"은 속도보다 품질을 우선시하므로 압축률은 높아지지만 인코딩 속도는 느려집니다. 속도 API 옵션 (아래) 과 상호 작용합니다. 기본값은 "좋음"입니다.

vp6_2패스_민_섹션 : 2패스 VBR 인코딩의 경우, 이 기능은 섹션이 아무리 쉬우더라도 인코더가 모든 섹션에 허용할 수 있는 가장 낮은 데이터 레이트를 제어합니다. 이 값은 어려운 섹션이 쉬운 섹션에서 너무 많은 비트를 훔치는 것을 방지하는 데 사용됩니다. 백분율로 표시되며 정수로 표시됩니다. 기본: 40.

예를 들어 video_bitrate 1000kbps로 설정되고 vp6_2_pass_min_section이 50으로 설정되면 섹션의 가장 낮은 비트 전송률은 500kbps (1000 * 50 %)입니다.

vp6_2패스_맥스_섹션 : 2패스 VBR 인코딩의 경우, 이 기능은 스트림할 수 있는 가장 높은 데이터 레이트와 섹션이 아무리 어려우더라도 인코더에서 허용하는 최고 데이터 레이트를 제어합니다. 기본: 400.

예를 들어 video_bitrate 1000kbps로 설정되고 vp6_2_pass_max_section은 450으로 설정되며 모든 섹션의 최고 비트 전송률은 4500kbps (1000 * 450 %)입니다.

vp6_스트림_프리버퍼 : 재생을 시작하기 전에 필요한 몇 초의 사전로드. 버퍼는 일관된 데이터 속도를 유지하고 재생 중단을 최소화하는 데 사용됩니다. 유효한상수 _ 비트 레이트인코딩 만. 기본: 6.

vp6_스트림_맥스_버퍼 : 버퍼의 최대 크기 (초)입니다. 유효한상수 _ 비트 레이트인코딩 만.

vp6_디인터랙스_모드 : 디인터레이싱이 활성화 된 경우이 옵션은 적용될 디인터레이싱 필터를 설정합니다. 유효한 옵션은 “적응형”, “흐림” (1:2:1 흐림 모드의 경우, “드롭” (필드 드롭 모드의 경우) 입니다. 기본값: 적응형.

vp6_디노이즈_레벨 : 소스의 노이즈 레벨을 지정합니다. 0보다 큰 값은 소스 내에서 일정하고 알려진 수준의 노이즈를 의미합니다. 값이 0이면 필터가 노이즈 레벨을 추정하여 프레임마다 다를 수 있습니다. 노이즈 레벨이 클수록 노이즈 소스가 더 많아 필터에 의한 제거가 증가하지만 선명도는 떨어집니다. 0에서 1 사이의 십진수 값이어야 합니다 (예: 0.2). 기본: 0.

fixed_keyframe_interval : 기본적으로 지정된 keyframe_interval은 두 키프레임 사이의 최대 프레임 수만 보장합니다. 인코더는 keyframe_interval에서 요구하기 전에 키 프레임을 배치하기로 결정할 수 있습니다. 이 옵션을 true로 설정하면 이러한 가변성이 방지됩니다. 기본값: false입니다.
