가져오기 API 키 가져오기

Zencoder API에 대한 모든 요청은 API 키를 사용하여 인증되어야합니다. 이 항목에서는 Zencoder 용 다양한 유형의 API 키와이를 얻는 방법에 대해 설명합니다.

API 키 가져오기

API 키를 요청하려면 Zencoder 사이트로 이동하십시오 . 이미 사용자로 등록한 경우 로그인하십시오. 그렇지 않은 경우 새 사용자로 등록하십시오. 사용자로 등록 할 때까지 API 키를 요청할 수 없습니다.

로그인 한 후 API 섹션으로 이동하십시오.

API 섹션에는 다양한 종류의 API 키를 요청하는 옵션이 표시됩니다.

다양한 유형의 API 키에 대한 설명은 아래 섹션을 참조하세요. 전체 액세스 키를 얻으려면 유료 요금제에 가입해야합니다.

전체 액세스 키

전체 액세스 키를 사용하면 Zencoder API에 모든 종류의 요청을 할 수 있습니다. 이것은 Zencoder에 작업을 제출하고 출력을 얻는 데 필요한 종류의 키입니다.

읽기 전용 키

읽기 전용 액세스 키를 사용하면 Zencoder API에만GET요청을 할 수 있습니다. 이는 작업 상태 만 가져 오면되지만 실제로 작업을 제출하지는 않는 사용자에게 유용합니다. 자세한 내용은 읽기 전용 API 키사용을 참조하십시오.

통합 모드 키

통합 모드 키를 사용하면 작업을 제출할 수 있지만 작업이 처리되지 않습니다. 주로 오류 요청을 테스트하려는 시스템 통합 자에게 유용합니다. Zencoder 유료 요금제를 구매하기 전에 API 사용에 대한 느낌을 얻거나 실제로 출력을 얻기 전에 요청을 확인하는 데 사용할 수도 있습니다.
