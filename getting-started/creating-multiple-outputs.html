다중 출력 생성

이 항목에서는 인코딩 작업에 대한 여러 출력을 만드는 방법에 대해 설명합니다.

여러 출력 생성

다음과 같이 출력 배열에 두 개 이상의 해시를 전달하여 여러 출력을 지정할 수 있습니다.

{
  "input": "https://user:pass@bucket-name/file-name.avi",
  "outputs": [
    {
      "label": "web",
      "url": "s3://output-bucket/web.mp4",
      "size": "512x384"
    },
    {
      "label": "iphone",
      "url": "s3://output-bucket/iphone.m4v",
      "size": "480x320"
    }
  ]
}
